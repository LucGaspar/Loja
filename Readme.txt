---------------------------------------------------------
INFORMAÇÕES
-----------
Sistema Operacional: Windows 8.1
IDE: NetBeans 8.0.2
=========================================================

---------------------------------------------------------
COMPILAÇÃO
----------
A compilação é natural: javac Cliente.java Servidor.java
=========================================================

---------------------------------------------------------
EXECUÇÃO
--------
A execução requer parâmetros

Servidor -> java -jar Servidor.jar PORTA
   Exemplo: java -jar Servidor.jar 1234

Cliente  -> java -jar Cliente.jar IP PORTA
   Exemplo: java -jar Cliente.jar 127.0.0.1 1234
=========================================================

---------------------------------------------------------
FUNCIONAMENTO
-------------
Servidor: já executado, basta escolher as opções
	  desejadas e inserir o que for requerido
	  Os arquivos CSV só serão atualizados se
          o servidor for fechado pela opção 5 - Sair

Cliente: já executado, escolha as opções, insira as
	 informações e aguarde os resultados
=========================================================
